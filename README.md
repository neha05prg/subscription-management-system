# subscription-management-system
Subscription Management System is a full-stack web application developed using Spring Boot that allows users to manage subscription plans through a secure and user-friendly interface. The application supports user authentication, role-based authorization, subscription tracking, and administrative management of users, plans, and subscriptions. It demonstrates enterprise-level backend development using Spring Boot, Spring Security, Hibernate/JPA, and MySQL.

Key Features
Secure user registration and login
Spring Security authentication and authorization
Role-based access (Admin & User)
Manage subscription plans
View active and expired subscriptions
Admin dashboard
CRUD operations for users and plans
Subscription management
MySQL database integration
Responsive web interface using Thymeleaf
Technologies Used

Spring Boot
Spring Security
Spring Data JPA (Hibernate)
