
# Subscription Management System

**Subscription Management System** is a full-stack web application developed using **Spring Boot**, **Spring Security**, **Hibernate/JPA**, **Thymeleaf**, and **MySQL**. The application provides a secure and efficient platform for managing subscription plans through role-based access control. It enables users to register, authenticate, view their subscriptions, and monitor active or expired plans, while administrators have complete control over managing users, subscription plans, and subscriptions.

The project follows the **Model-View-Controller (MVC)** architecture and demonstrates enterprise-level backend development practices, including secure authentication, authorization, database integration, layered architecture, and responsive web design. It serves as an excellent example of building scalable and maintainable Java web applications using modern Spring technologies.

---

## Key Features

### User Module

* User registration and secure login
* Authentication using Spring Security
* View active and expired subscriptions
* View subscription plan details
* Personalized user dashboard
* Secure logout functionality

### Admin Module

* Admin dashboard with system overview
* Create, update, and delete subscription plans
* View all registered users
* Update and delete user accounts
* Manage user subscriptions
* Delete subscriptions
* Role-based administrative privileges

### Security Features

* Secure authentication using Spring Security
* Password encryption using BCrypt
* Role-based authorization (ADMIN & USER)
* Protected routes and restricted access
* Secure session management

### Database Management

* MySQL database integration
* Hibernate/JPA for Object Relational Mapping (ORM)
* Repository pattern using Spring Data JPA
* Automatic seed data initialization
* Efficient CRUD operations

### User Interface

* Responsive web pages using Thymeleaf
* HTML5, CSS3, and JavaScript
* Interactive dashboards
* Clean and intuitive user experience
* Real-time notification toasts
